
# 📊 Bank Marketing Funnel Analysis

**Future Interns — Task 3**

An end-to-end data analytics project analyzing a bank telemarketing campaign to understand what drives customers to subscribe to a term deposit.

---

## 🛠️ Tools Used
- **Python** — Data cleaning
- **SQL Server** — Data storage & analysis queries
- **Power BI** — Interactive dashboard

---

## 📁 Dataset
- 39,178 customer records
- 25 features (demographics, campaign details, outcomes)
- Target: Did the customer subscribe? (yes/no)

---

## 📊 Dashboard Pages
1. **KPI Overview** — Total contacts, conversion rate, avg call duration
2. **Funnel Analysis** — Conversion by call frequency
3. **Demographics** — Subscription by age, job, education
4. **Call Analysis** — Call duration vs conversion rate
5. **Monthly Trends** — Campaign performance over time

---

## 🔍 Key Findings
- Overall conversion rate: **11.13%**
- Very long calls (10+ min) convert at **49%** vs 1% for short calls
- Previous campaign success = **65% conversion rate**
- Students and retired customers convert the most
- Best months: **March, September & October**
